# Техническое задание на разработку визуальных и картографических материалов для сайта мастер-плана Твери

**Дата составления:** 2025-01-27  
**Версия:** 1.0  
**Проект:** Мастер-план города Твери и агломерации на 2025–2035 годы и на период до 2045 года

---

## 1. Общие требования

### 1.1. Назначение документа
Данное ТЗ определяет требования к разработке визуальных материалов (фотографии, иллюстрации, рендеры) и картографических материалов (ГИС-слои, схемы, карты) для презентационного сайта мастер-плана Твери.

### 1.2. Технические требования к файлам

#### Изображения
- **Формат:** JPEG (для фотографий), PNG (для схем и графики с прозрачностью), WebP (опционально, для оптимизации)
- **Разрешение:**
  - Hero-изображения: минимум 1920×1080px (Full HD), рекомендуется 3840×2160px (4K)
  - Карточки и иллюстрации: минимум 1200×800px
  - Иконки и мелкие элементы: 512×512px (SVG предпочтительно)
- **Качество:** JPEG сжатие 85-90% для фотографий, без потерь для схем
- **Цветовое пространство:** sRGB
- **Названия файлов:** латиница, нижний регистр, с подчеркиваниями (например: `tver_hero_volga_view.jpg`)

#### Видео
- **Формат:** MP4 (H.264)
- **Разрешение:** 1920×1080px (Full HD)
- **Битрейт:** 5-8 Mbps
- **Длительность:** 10-30 секунд для фоновых видео
- **Звук:** без звука (muted) для фоновых видео

#### Картографические материалы
- **Формат:** PNG (для экспорта), исходные файлы в формате QGIS/MapInfo
- **Система координат:** МСК-69 (местная система координат)
- **Разрешение:** минимум 300 DPI для печатных материалов, 150 DPI для веб-версии
- **Размеры:** адаптивные, но не менее 2000×1500px для основных карт
- **Стилизация:** единый стиль согласно дизайн-киту проекта

## 3. Картографические материалы (ГИС-слои)

### 3.1. Общие требования к картографическим материалам

#### 3.1.1. Технические параметры
- **Система координат:** МСК-69 (местная система координат Тверской области)
- **Масштаб:** 1:10 000 - 1:25 000 для городских карт, 1:50 000 для агломерации
- **Формат исходных данных:** QGIS (.qgs) или MapInfo (.tab)
- **Формат экспорта для сайта:** PNG (веб-версия), PDF (печатная версия)
- **Разрешение:** 300 DPI для печати, 150 DPI для веб
- **Размеры:** адаптивные, но не менее 2000×1500px

#### 3.1.2. Стилизация карт
- **Цветовая палитра:** согласно дизайн-киту проекта
  - Heritage (медный/золотой): `#c27a35`
  - Water (бирюзовый): `#0f766e`
  - Digital (синий): `#2563eb`
  - Графит (темно-серый): `#1f2933`
- **Типографика:** Montserrat для заголовков, Inter для подписей
- **Легенда:** обязательна для всех карт
- **Масштабная линейка:** обязательна
- **Ориентация (север):** обязательна

## 6. Требования к качеству и стилю

### 6.1. Фотографии

- **Стиль:** Профессиональная архитектурная и городская фотография
- **Освещение:** Естественное, предпочтительно "золотой час" (рассвет/закат)
- **Композиция:** Учитывать правило третей, ведущие линии
- **Обработка:** Минимальная цветокоррекция, сохранение естественных цветов
- **Лицензия:** Должны быть права на использование (собственная съемка или лицензионные материалы)

### 6.2. 3D-рендеры

- **Стиль:** Фотореалистичные визуализации
- **Детализация:** Высокая, с проработкой материалов, освещения, окружения
- **Ракурсы:** Несколько ракурсов для ключевых проектов (общий вид, детали, интерьеры)
- **Окружение:** Реалистичное окружение с учетом существующей застройки

### 6.3. Картографические материалы

- **Стиль:** Современный, минималистичный, читаемый
- **Цветовое кодирование:** Согласно дизайн-киту проекта
- **Легенда:** Понятная, с иконками и подписями
- **Типографика:** Четкая, читаемая на всех размерах
- **Интерактивность:** Возможность масштабирования (для веб-версии)
